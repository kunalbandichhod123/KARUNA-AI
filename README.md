## ABOUT PROJECT :
KARUNA-AI is a highly advanced chat and voice assistant dedicated to Ayurveda. With a futuristic UI design, smart session memory, and state-of-the-art AI integration, it provides accurate and conversational health tips. Give it a try.

## What makes KARUNA-AI so powerful :
* **Smart Two-Level LLM Architecture:**
  * **70B Versatile Model:** Handles comprehensive chat queries and processes context through RAG for highly detailed and accurate answers.
  * **8B Instant Model:** In voice mode, it takes the 70B's detailed output and creates a highly concise version to save TTS tokens, giving a natural human-like conversational feel, while keeping the full answer visible in the chat interface.
* **Advanced RAG Engine:** Minimizes hallucinations using a robust pipeline of document processing and intelligent context retrieval from highly-curated Ayurvedic PDFs.
  * **SHA-256 Hashing & Smart Chunking:** Uses NLP-based block detection (NLTK) to chunk Ayurvedic literature by specific headings (e.g., Nidana, Chikitsa) while applying SHA-256 deduplication at ingest.
  * **Hybrid Retrieval:** Blends Semantic Search (FAISS vector indices via `sentence-transformers`) and Keyword Search (Whoosh BM25) to prioritize intent without losing granular Ayurvedic terminology matches. 
  * **Dynamic Validation & Food Randomization Logic:** Dynamically adjusts retrieval depth (`top_k`) and applies smart randomization for lifestyle/diet queries (using tools like `rapidfuzz`), ensuring dietary recommendations are diverse and don't feel repetitive, while keeping medical answers strictly deterministic.
* **Smart Memory:** Maintains conversation context intelligently using a session store.
* **Futuristic UI Design:** Aesthetic, responsive, and immersive frontend experience.

## INSTALLATION & USAGE :

### Run Simple Streamlit UI

1. **Install dependencies :**
```bash
pip install -r requirements.txt
```

2. **Activate the virtual environment :**
```bash
conda activate venv/
```

3. **Change directory to core backend :**
```bash
cd "BACKEND\core"
```

4. **Run the Streamlit UI :**
```bash
streamlit run app.py
```

### Run Advanced & Aesthetic UI

*NOTE: Make sure you have Node installed on your PC. Check your version with `node -v`. If not installed, download and install Node.js.*

1. **Change directory to frontend :**
```bash
cd "FRONTEND\vedabuddy complete ui\ayur-voice-guide-main"
```

2. **Install frontend dependencies :**
```bash
npm install
```
*(Description: This command reads the package.json file, downloads all required packages, and places them in the node_modules folder.)*

3. **Build the production folder :**
*(If your frontend folder does not have a folder named `dist`, run this command)*
```bash
npm run build
```
*(Description: This creates a new folder named `dist` inside ayur-voice-guide-main, which will be served via FastAPI.)*

4. **Go back to your main folder :**
```bash
cd "..\..\.."
```

5. **Navigate into the core folder from BACKEND :**
```bash
cd "BACKEND\core"
```

6. **Run the final code file :**
```bash
python main.py
```
*(Description: This runs our main FastAPI backend application.)*

## TECH STACK & ARCHITECTURE
* **Backend:** FastAPI, Python
* **Frontend:** React, Node.js, Streamlit
* **AI & LLM:** Groq, LangChain, 70B Model (Chat/RAG), 8B Model (Voice Truncation)
* **Search & Indexing:** FAISS (Vector DB), Whoosh (Keyword Index), Sentence-Transformers (`all-MiniLM-L6-v2`), Cross-Encoder Reranking
* **Text Processing Pipeline:** NLTK, SHA-256 deduplication, RapidFuzz
* **Data Source:** High-quality Ayurvedic PDFs
